サンプル

ロールオーバー時の読み込みを軽減させるための、プリロード付きロールオーバーJavaScriptです。
Develoop ToolのNetworkからロールオーバー後の画像となる「button01_over.png」がマウスオーバー前に読み込まれていることがわかります。

使い方は簡単で対象となる要素を参照し、プラグインを実行するだけです。
オプションでサフィックスの変更が可能です。 

<a href="#"><img src="button01.png" width="180" height="57" alt="button"></a>
<a href="#"><img src="button01_out.png" width="180" height="57" alt="button"></a>
<a href="#"><img src="button01_off.png" width="180" height="57" alt="button" class="rollover"></a>
<script src="jquery.easy-rollover.js"></script>
<script>
$(function(){
	// 通常のサンプル
	$('img').easyRollover();

	// サフィックスを変更したサンプル
	$('.rollover').easyRollover({
		suffix: '_off.',
		suffix_replace: '_on.'
	});
});
</script>
